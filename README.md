  MeloDash: Audio Experience Design
	
## Introduction

Melodash is a running game based on real time music analyzed. Player’s objective is to dodge obstacles that are automatically generated by the music. Our group’s focus was to develop the game further by completing beat detection, and adding extra features such as music visualisation.
